# RecHERsion 2026

RecHERsion 2026 is an all-girls coding event hosted collaboratively by:

- ACM-W NIT Karnataka
- ACM-W BPDC (BITS PILANI DUBAI CAMPUS)
- ACM RIT

## Event Theme

**Everyday Chaos, Algorithm Solutions**

## Event Dates

- **18 April 2026, 10:00 AM IST**: Contest begins
- **19 April 2026**: Winner announcement


## Website Highlights

- Purple-themed interactive frontend with animated hero and theme highlight
- Live countdown to contest start
- Chapter logos with website links
- Event timeline and FAQ section
- Registration form with validation for participant details
- Registration storage via Vercel API + Google Sheets (service account)

## Project Files

- `index.html`: Current event website
- `api/register.js`: Serverless API route that writes submissions to Google Sheets
- `package.json`: Project dependency config (`googleapis`)
- `code.html`: Legacy/alternate version retained in the repo
- `HOSTING_SETUP.md`: Setup guide for Vercel + Google Sheets service account flow
